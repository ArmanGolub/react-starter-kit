# KNPP React Starter 🚀

A modern **Vite + React + TypeScript** starter packed with everything you actually use in real projects:

- ⚡️ Vite + SWC (fast dev & builds)
- 🎨 Tailwind CSS 3 + shadcn/ui components
- 💫 Framer Motion animations
- 🌍 i18next (kk / en / ru) with language detection & lazy-loaded JSON
- 🧠 Zustand for global state
- 🧾 React Hook Form for forms
- 🧭 React Router for SPA navigation
- 🧹 ESLint 9 + Prettier + import sorting + Tailwind plugin

This repo is a solid base for dashboards, internal tools, and CMS-style apps.

---

## ✨ Features

- **Beautiful Home page** showing off the tech stack & a Zustand-powered counter.
- **About page** with a **react-hook-form**-powered contact form.
- **Multilingual** (Kazakh / English / Russian) using i18next, with auto language detection & localStorage.
- **Design system friendly** via shadcn/ui and Tailwind tokens (semantic colors, radii, etc.).
- **Aliases & clean imports** via `@` → `src`.
- Production-ready **linting & formatting** setup.

---

## 🧱 Tech Stack

- **Core:** React 19, TypeScript, Vite 7
- **Routing:** `react-router-dom`
- **State:** `zustand`
- **Forms:** `react-hook-form`
- **Styling:** Tailwind CSS 3, `tailwindcss-animate`
- **UI components:** `shadcn/ui` (generated into `src/common/components/ui`)
- **Animations:** `framer-motion`
- **Icons:** `lucide-react`
- **i18n:** `i18next`, `react-i18next`, `i18next-browser-languagedetector`, `i18next-http-backend`
- **Tooling:** ESLint 9 (flat config), Prettier 3, import sorting, Tailwind Prettier plugin

## 📦 Getting Started

### Prerequisites

- **Node.js** ≥ 18 (recommended)
- **Yarn** (or npm, if you prefer)

### Install dependencies

```bash
yarn install
# or
npm install
```
